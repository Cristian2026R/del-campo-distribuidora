DON VALENTIN - versión con icono personalizado

Subir a GitHub:
- app.py
- requirements.txt
- lista_don_valentin.xlsx
- assets/ completa

Después de subir:
1. Esperar 1-2 minutos.
2. Borrar el acceso directo viejo del escritorio.
3. Cerrar Chrome completo.
4. Abrir la app nuevamente.
5. Crear un acceso directo nuevo marcando 'Abrir como ventana'.
