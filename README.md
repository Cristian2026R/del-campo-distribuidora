# DON VALENTIN V7

Cambios:
- Ticket con precio automático por defecto para gramos, litros y unidad.
- Precio y total siguen siendo editables manualmente antes de agregar al ticket.
- Proveedores: al seleccionar un proveedor, se puede registrar pago manual al proveedor con producto/concepto, cantidad, unidad, precio unitario, total y medio de pago.
- Cada pago al proveedor se registra automáticamente como egreso de caja y en reportes históricos.
